Project Overview

The AI Data Analyst app allows users to upload CSV datasets and automatically:

Provides a complete overview of the dataset.

Suggests data cleaning steps.

Generates interactive visualizations.

Produces AI-generated insights in plain English.

Helps users explore trends, correlations, and anomalies without writing code.

This project is ideal for data analysts, business professionals, and AI enthusiasts who want to quickly understand their data.

Features

Data Overview: Display first rows, data types, missing values, statistics.

Data Cleaning Suggestions: Detect missing or duplicate values and suggest fixes.

Visualizations: Histograms, boxplots, correlation heatmaps, and bar charts for categorical columns.

AI Insights: Generate plain-language insights, trends, and anomalies.

Interactive Filtering: Select columns and filter data for customized analysis.

Optional Prediction: Suggest predictions if a target column is present.

Tech Stack

Python 3.x

Streamlit – Frontend web interface

Pandas / NumPy – Data processing

Matplotlib / Seaborn / Plotly – Visualizations

Optional: OpenAI API for natural language insights

Installation

Clone the repository:

git clone https://github.com/yourusername/ai-data-analyst.git
cd ai-data-analyst

Install required packages:

pip install -r requirements.txt

Run the app:

streamlit run app.py
Usage

Open the app in your browser (Streamlit will provide a link).

Upload a CSV dataset.

Explore the data overview, cleaning suggestions, visualizations, and insights.

Interact with filters to customize your analysis.
